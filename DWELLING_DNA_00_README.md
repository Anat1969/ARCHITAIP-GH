# DWELLING DNA — README
## הקובץ הראשון שקלוד קוד קורא בכל session חדש

---

### מה הפרויקט
מגזין דיגיטלי חי שמתרגם DNA תרבותי למרחבי מחייה ויזואליים.
לא אתר שמתחפש למגזין — מגזין אמיתי שמתפקד כאתר.
משתמש יחיד: אדריכלית-אומנית-חוקרת-מרצה.
פלט: ללקוחות, כנסים, הרצאות.

---

### שבעת המסמכים — מה כל אחד מכיל

```
01_Product_Brief        — מה הכלי, למי, מה הוא לא עושה. קרא ראשון.
02_Data_Architecture    — מבנה הנתונים המלא: תרבות, עמוד, גרסה, תמונה.
03_Screens_and_Flow     — כל מסך, מה הוא מציג, איך מנווטים.
04_Prompt_Engine_Logic  — הלוגיקה שמייצרת פרומפטים מ-DNA תרבותי.
05_Content_Seed         — 11 תרבויות מוכנות כ-JSON מלא.
06_Design_Vision        — חוקי השפה הגרפית, פונטים, פלטות, פורמטי עמוד.
07_Image_System         — API, גלריה, שמירה, כפתור פרומפט, אינטגרציה.
```
