### מסך 1 — Cultures Home
**מסך הכניסה הראשי**

תצוגת גריד של כל התרבויות.
כל כרטיס תרבות מציג:
- שם התרבות (גדול, בפונט ייחודי)
- תמונה מייצגת כרקע
- פלטת הצבעים של התרבות
- מספר עמודים קיימים
- תאריך עדכון אחרון

פעולות מהמסך:
- לחיצה על תרבות → כניסה למגזין
- כפתור "הוסף תרבות" → modal יצירה
- ניווט לגלריה הגלובלית

---

### מסך 2 — Culture / Magazine
**הפרק הפנימי של כל תרבות**

ניווט עליון: שם התרבות + שלושה פרקים (פרטי / מעבר / קהילתי)
תצוגת עמודים: גריד של כל העמודים בפרק הנוכחי

כל כרטיס עמוד מציג:
- תמונה ראשית
- כותרת
- מספר עמוד
- אינדיקטור גרסאות

פעולות:
- לחיצה על עמוד → כניסה לעורך
- כפתור "עמוד חדש" → עורך ריק
- מעבר בין פרקים
- צפייה בגלריה של התרבות

### מסך 3 — Page Editor
**לב הכלי**

שני מצבים: עריכה / הצגה

**מצב עריכה:**

```
┌─────────────────────────────────────────────┐
│  TOOLBAR (עליון)                             │
│  שמור גרסה | ייצא | מצב הצגה              │
├─────────────────┬───────────────────────────┤
│                 │  EDIT PANEL (ימין)         │
│   PAGE CANVAS   │  ─────────────────────    │
│   (מרכז)        │  טיפוגרפיה                │
│                 │  פריסה                     │
│  עריכה ישירה   │  שכבות                     │
│  על כל אלמנט   │  תמונות                    │
│                 │  צבעים                     │
│                 │  גרסאות                    │
└─────────────────┴───────────────────────────┘
```

עריכה ישירה על הקנבס — לחיצה על כל אלמנט עורכת אותו במקום.
פאנל ימין — כלים מתקדמים: שכבות, היררכיה, פריסה, ניהול תמונות.

**מצב הצגה:**
הממשק מתקפל לסרגל דק בקצה המסך.
לחיצה אחת — הממשק חוזר.
הסרגל מכיל: חץ קדימה/אחורה, מספר עמוד, כפתור יציאה.

### מסך 4 — Global Gallery
**מסך עצמאי, נגיש מכל מקום**

תצוגת גריד של כל התמונות — מכל התרבויות.

פילטור:
- לפי תרבות
- לפי פרק (פרטי / מעבר / קהילתי)
- לפי מקור (generated / uploaded / pasted)
- לפי גרסה

כל כרטיס תמונה מציג:
- התמונה
- שם התרבות
- כפתור "הצג / העתק פרומפט"
- פעולות: הוסף לעמוד | מחק | שכפל

### מסך 5 — Add Culture (Modal)
**חלון הוספת תרבות חדשה**

שלב א — שם התרבות (קלט טקסט)
שלב ב — המערכת מנתחת אוטומטית:
  - כאוס, סדר, תפקיד אנושי
  - שישה וקטורי DNA
  - פלטת צבעים
שלב ג — תצוגת תוצאות לאישור
שלב ד — אישור / דחייה / עריכה ידנית
